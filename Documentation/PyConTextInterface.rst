PyConText Inteface
******************

This group of modules provides tools for working with PyConText. The main module is PyConText which can be used to run the PyConText algorithm and return Document instances. There are also two helper modules, SentenceReconstructor and SentenceRepeatManager. These two modules are utilized internally by the PyConText module but documentation for them is included here.
